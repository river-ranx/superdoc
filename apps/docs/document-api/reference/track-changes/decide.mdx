## Summary

Accept or reject a tracked change (by ID or scope: all).

- Operation ID: `trackChanges.decide`
- API member path: `editor.doc.trackChanges.decide(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a Receipt confirming the decision was applied; reports NO_OP if the change was already resolved.

## Input fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `decision` | enum | yes | `"accept"`, `"reject"` |
| `target` | object \\| object | yes | One of: object, object |

### Example request

```json
{
  "decision": "accept",
  "target": {
    "id": "id-001",
    "story": {
      "kind": "story",
      "storyType": "body"
    }
  }
}
```
