# Moved to demos/editor/custom-ui

The custom-UI reference workspace moved to [`demos/editor/custom-ui`](../editor/custom-ui).

It now sits under `demos/editor/` to mirror the docs nav (Editor > Custom UI). The workspace is a reference for composing many SuperDoc UI surfaces (toolbar, comments, tracked changes, citations, custom commands, context menus) in one app, including the source-grounded citation flow.
